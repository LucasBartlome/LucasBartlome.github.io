Project 1: AI Podcast Generator

Worked with a team at Motivation Technology to create an AI-based Podcast Generator prototype loosely based on Google’s NotebookLM project. The project used Python within a VS Code environment, and OpenAI’s ChatGPT and TTS (Text to Speech) API call to create the Podcast scripts. A simple UI was created to allow granular control of the parameters used to create podcasts (e.g. Title and Episode, names of hosts, voices used). The output was an MP3 file that could be played in a simple web-based player.

  • OpenAI GPT and Text-to-Speech APIs
  • Python and Visual Studio Code
  • Use AI to create podcast script using GPT APIs and Outputs in JSON format
  • Took advantage of OpenAIs TTS to create mp3 fragments

After running the program it gives you an MP3 File of your fully completed AI generated Podcast!

Project 3: DataBase In MySQL

I lead a group in which we created a database in mysql for a fake company we called phoenix corp to keep track of all employees and products through their store to identify patterns and keep track of all of their information. We had roughly 10 entities that were used including 2 sets of inherited entities for employee type as well as for product type. We then created an entity relationship diagram to help us to create the database itself as well as get a better idea of how it can be used it in the long run. We then populated the database with fake information and created 15 different queries in MySQL using a combination of left and right joins to find specific instances and cases within our data that we made. I really enjoyed this project because it gave us an insight into how a company would use a database and writing useful SQL queries.
