Overview:
Bit Fishing is a 2D fishing simulation game developed entirely in Python using the Pygame and PyTMX libraries. The project combines sprite animation, physics, and map-based world exploration to create a relaxing yet progression-driven fishing experience. Players begin at the farm and can unlock new fishing locations such as the Pond, River, Lake, and Ocean ‚Äî each featuring unique fish species and challenges.

Core Technologies:

  • Python with Pygame for core engine, animation, and input handling.
  • PyTMX and Tiled Map Editor (.tmx) for creating interactive worlds with layered maps and collision tiles.
  • Pillow (PIL) for GIF and sprite sheet animation support.
  • JSON for storing save data and defining fish tables (species, size, rarity, and value).
  • Object-oriented architecture for reusable classes (Scenes, Overlays, Camera, Player, Boat, etc.).

Project 3: Database In MySQL

Collaborated on a team project to design and implement a database in MySQL for a test company called Phoenix Corp. The goal was to track all employees and products in a simulated store environment in order to identify meaningful patterns within the data. We started by designing an entity-relationship diagram (ERD) to map out the structure of the database, which included roughly 10 entities. Two of these sets were modeled using inheritance‚Äîone for different employee types and another for different product types. Using the ERD as a guide, we built the database schema and populated it with realistic testing data. To demonstrate the value of the system, we developed 15 SQL queries that incorporated joins, aggregate functions, subqueries, and more to extract insights and highlight specific business cases within the data. This experience strengthened my skills in relational database design and advanced query writing, while also giving me valuable perspective on how businesses leverage databases to analyze operations and drive data-driven decisions.
